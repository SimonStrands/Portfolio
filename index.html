Game Engine DirectX 11

expand_more

A game engine created in C++ and DirectX11.
The engine uses multiple techniques such as:
Deffered Rendering: To handle multiple dynamic lighs faster.
Custom Obj File loader: The engine has an custom obj file loader that can load different models and meshes with different kind of materials.
Displacement Mapping and level of detail using Tessellation Hardware: Based on the cameras position, some objects will transform and get smoother edges or change the vertecies position based on a height map.
Dynamic Shadow Mapping: To create "realistic" shadows.
Dynamic Cube mapping: Taking 6 screenshots of the scene each frame to create a 360Â° mirror of the scene
View Frustum Culling using Quadtree: To cull away static objects that the camera is not looking at
Small particle system: Using points in a vertecies buffer to create billboarded particles and a custom compute shader to update the position of the particles to create a small particle system.
